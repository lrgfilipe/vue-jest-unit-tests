
# VueJs + Jest (Simple-unit-tests)

## Project setup
```
npm install
npm run serve
```

### Run unit tests
```
npm run test:unit
```

 - [X] clicking button "increment" increments the counter value by 1
 - [X] clicking button "reset" resets the counter to "0"
 - [X] clicking button "decrement" decrements the counter value by 1
